Firing Range Club Membership Website

Created in Angular 2, using Firebase and implements CSS, HTML, and Javascript technologies.

Intended Use

Created as an eventual replacement to the current club website. A more responsive and useful website was desired for the needs of the Club managers. Visualizing and being able to actually see database information (Members, payment dues) were the main reasons for creation of the replacement. Previously, little to no querying could be done to view useful information about members. It was difficult to determine who actually was a member, and who actually had paid their regular membership fees.

Features

  • Visually pleasing and responsive interface
  • Ability for privileged users to push new notifications to the home-page as well as monitor user records.
  • Secure navigation/Traversing through pages(Accomplished through Angular's routing using guards)
  • Database integration to allow for searching and viewing of member information

Senior Project (Screenshot Tagger Program)

Three part project. Lua and XML Addon written for World of Warcraft, local application written in Java, and web-interface written in Angular with sails.js

Poster of the project

Intended Use

A team of sociologists submitted a request for a program that would allow for better behavoral study of events in an online environment, specifically World of Warcraft. It was desired to create a platform where researchers could catalog and more easily analyze/share events from in-game.

Features

  • Lua Addon that allows for taking of screenshots as well as record keeping of the events being depicted in the screenshots themselves. Chat log dumps are also used.
  • User client written in Java allows for the preparation of all data from the Addon to be submitted to the online web-system
  • Web front-end written in Angular allows for querying and searching of data between multiple users.

Android/PHP Book Store System

Created in Android Studio, using Java and PHP for SQL database interfacing. Site made using basic CSS, HTML, and Javascript technologies.

Login screen and search results retrieved from the DB during a query

Admin panel where more powerful options are available for manipulating the database/viewing entries

Intended Use

Created to simulate an Android bookstore application where users can register for a store, search and buy books. Users are able to search, purchase, and view the history of their purchases

Features

  • mySQLi database integration to accomodate for the storage and retrieval of users, books, and purchase history
  • Admin panel where new books may be added into the database as well as viewing of all user settings/purchase history

Taiga C++ Addon

Built off the open source program Taiga to fulfill a personal need for persistent playlists of television shows. Developed in Visual Studio 2017 using C++ and some XML elements for UI display. Gained experience working with an open source project and mimicking pre-existing coding conventions as well as re-using already created functions.

Newly created area on the program along with the listview of the shows in the playlist.

Generated playlist file.

Intended Use

Previously every show needed to be started individually, and consecutive playback was not possible. Main motivation was to allow for this type of behavior in the program.

Features

  • Drag/dropping of listview titles to reorganize the playlist to the user's choosing.
  • Automatically updates user file in local storage for persistence through program restarts.
  • Exportable dynamic playlist in M3U support.
