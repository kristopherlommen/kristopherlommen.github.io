Firing Range Club Membership Website

Created in Angular 2, using sails.js and implementing CSS, SQL, HTML, and Javascript technologies.

Intended Use

Created as a replacement to the previous club website. A more responsive and useful website was desired for the needs of the Club managers. Visualizing and being able to actually see database information (Members, payment dues) were the main reasons for creation of the replacement. Previously, little to no querying could be done to view useful information about members. It was difficult to determine who actually was a member, and who actually had paid their regular membership fees.

Features

  • Visually pleasing and responsive interface
  • Secure navigation/Traversing through pages(Accomplished through Angular's routing using guards)
  • Database integration to allow for searching and viewing of member information
Planned Features
  • More abilities for Administrators (Pre-built queries such as viewing members that are due to pay/soon due to pay)
  • Dynamic Output of DOM (Pushing new alerts/notifications to the site as well as overall customization)
  • Increased ability for a person, regardless of technological background, to dynamically change the site's interface to their needs

Android/PHP Book Store System

Created in Android Studio, using Java and PHP for SQL database interfacing. Site made using basic CSS, HTML, and Javascript technologies.

Login screen and search results retrieved from the DB during a query

Admin panel where more powerful options are available for manipulating the database/viewing entries

Intended Use

Created to simulate an Android bookstore application where users can register for a store, search and buy books. Users are able to search, purchase, and view the history of their purchases

Features

  • mySQLi database integration to accomodate for the storage and retrieval of users, books, and purchase history
  • Admin panel where new books may be added into the database as well as viewing of all user settings/purchase history
